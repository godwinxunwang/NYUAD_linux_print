# NYUAD_linux_print
NYUAD pharos printer support for Linux

*****
README
*****
Printing on NYU Abu Dhabi printers from Ubuntu (Deb) machines.

*****
INSTALL
*****
Straighforward, just ```source NYUAD_UNX_PRINT.sh```
